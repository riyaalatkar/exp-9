### Theory: 
NumPy (Numerical Python) is a fundamental library in Python used for numerical computations and handling large, multi-dimensional arrays and matrices. 
It provides an efficient and flexible way to perform mathematical operations on data, making it a cornerstone of data science, machine learning, and scientific computing in Python.

1. np.array(): Used to create one-dimensional or multi-dimensional arrays.
2. ndim: Returns the number of dimensions of an array.
3. shape: Returns the size of the array in terms of rows and columns.
4. dtype: Displays the datatype of elements stored in the array.
5. np.zeros(): Creates an array filled with zeros of a specified size.
6. np.ones(): Creates an array filled with ones.
7. np.eye(): Generates an identity matrix.
8. np.arange(): Generates evenly spaced values within a given range using start, stop, and step size.
9. np.linspace(): Generates evenly spaced numbers between two values with a specified number of elements.

Array arithmetic: NumPy allows element-wise addition, subtraction, multiplication, and division.

1. np.mean(): Calculates the average value of array elements
2. np.median(): Returns the median value of array elements.
3. np.max() / np.min(): Returns maximum and minimum values from the array.
4. np.sum(): Calculates the sum of all elements in the array.
5. np.transpose(): Transposes rows into columns in a matrix.
